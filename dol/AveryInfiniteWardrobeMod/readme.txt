=======================================
无限衣柜 Mod - 全地点版 v1.2.0
Infinite Wardrobe Mod - All Locations
=======================================

【功能说明】
将所有衣柜的容量扩展为 99999 件，
再也不用担心衣服放不下了！

【版本更新】
v1.2.0 - 修复生效问题
- 添加了多种方式确保容量修改生效
- 使用 JavaScript + Twee 双保险
- 在每个 passage 自动检查并修改

【安装方法】
1. 启动游戏，打开 Mod 管理器（左下角按钮或按 Alt+M）
2. 删除旧版本的无限衣柜 Mod（如果有）
3. 点击"选择文件"，选择 AveryInfiniteWardrobe.mod.zip
4. 点击"添加旁加载 Mod"
5. 看到 "Success. 刷新页面后生效" 后，点击"重新载入"
6. 进入游戏，打开衣柜查看容量

【验证是否生效】
1. 进入游戏后，按 F12 打开浏览器控制台
2. 查看是否有 "[无限衣柜Mod]" 开头的日志
3. 打开衣柜，查看容量显示是否为 99999
4. 尝试购买或存放大量衣服，看是否还能继续存放

【如果还是不生效】
1. 确保游戏版本较新（支持 ModLoader v2.0+）
2. 尝试先禁用其他 Mod，只保留这个测试
3. 检查控制台是否有红色错误信息
4. 尝试刷新页面后再次打开衣柜

【技术细节】
- 修改了 $wardrobe.space 变量
- 重写了衣柜检查函数
- 使用定时器持续监控并修复
- 适用于所有衣柜场景
