=======================================
艾弗里豪宅无限衣柜 Mod
Avery Mansion Infinite Wardrobe Mod
=======================================

【功能说明】
当玩家入住艾弗里的豪宅后，该地点的衣柜容量将变为无限（实际上是999999件），
再也不用担心衣服放不下了！

【安装方法】
1. 启动游戏，打开 Mod 管理器（左下角按钮或按 Alt+M）
2. 点击"选择文件"，选择 AveryInfiniteWardrobe.mod.zip
3. 点击"添加旁加载 Mod"
4. 看到 "Success. 刷新页面后生效" 后，点击"重新载入"
5. 进入游戏，当你入住艾弗里豪宅后，衣柜将自动变为无限容量

【检测条件】
本 Mod 通过以下方式检测玩家是否在艾弗里豪宅：
- 检查当前所在 passage 是否包含 "Avery", "Mansion", "Condo" 等关键词
- 检查变量 $avery.condo / $avery.mansion 是否为 true
- 检查 $livingLocation / $home 是否包含艾弗里豪宅相关信息

【兼容性】
- 需要 ModLoader v2.0.0 或更高版本
- 与大多数 Mod 兼容
- 不会修改游戏存档

【技术细节】
- 修改了 wardrobe.space 变量
- 重写了 wardrobe 满检查函数
- 仅在艾弗里豪宅区域生效，离开后会恢复正常
